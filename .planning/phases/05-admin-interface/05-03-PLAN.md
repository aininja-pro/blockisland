**Premium logic from Phase 3:**
- is_premium boolean flag on listings
- rotation_position integer (1 = first in rotation)
- last_rotated_at timestamp
- Daily rotation moves position 1 to end, shifts others up
- Per-category rotation (each category rotates independently)

**Key insight:** Chamber staff need to:
1. See who is premium at a glance
2. Toggle premium status easily
3. Understand the current rotation order
4. See when rotation last occurred
</context>
