**Tech stack decisions:**
- Express.js — Simple, well-documented API framework
- Supabase — Managed Postgres + built-in auth (free tier covers needs)
- @supabase/supabase-js — Official client library
- Deploy target: Render (~$10-20/month for Express, Supabase free)

**Why Supabase over SQLite:**
- Built-in auth simplifies Phase 5 admin interface
- Free tier generous (500MB, unlimited API)
- Managed = less ops burden, fits "set it and forget it"
- Real Postgres for future flexibility

**Key constraints:**
- Must support lat/long coordinates for map pins
- Must support categories with independent rotation per category
- Premium rotation applies within each category
- ~30-40 premium members, premium status changes annually
</context>
