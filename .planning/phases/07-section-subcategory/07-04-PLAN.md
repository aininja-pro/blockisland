Update the categories page to display the section/subcategory hierarchy:

1. Update categories.ts to query by section:
   ```typescript
   export interface SectionStats {
     section: string
     totalCount: number
     premiumCount: number
     subcategories: {
       name: string
       count: number
     }[]
   }

   export async function getSectionStats(): Promise<SectionStats[]> {
     const supabase = await createClient()

     // Get all listings with section
     const { data: listings, error } = await supabase
       .from('listings')
       .select('section, is_premium')

     if (error) throw error

     // Group by section
     const stats: Record<string, { total: number; premium: number }> = {}
     for (const listing of listings || []) {
       if (!listing.section) continue
       if (!stats[listing.section]) {
         stats[listing.section] = { total: 0, premium: 0 }
       }
       stats[listing.section].total++
       if (listing.is_premium) {
         stats[listing.section].premium++
       }
     }

     return Object.entries(stats)
       .map(([section, counts]) => ({
         section,
         totalCount: counts.total,
         premiumCount: counts.premium,
         subcategories: [] // Optional: can add subcategory breakdown later
       }))
       .sort((a, b) => a.section.localeCompare(b.section))
   }
   ```

   - Display sections instead of categories
   - Update card titles from "Category" to "Section"

3. Keep it simple - just change the grouping from category to section. Don't add complex subcategory breakdowns (YAGNI for 2-3 admin users).
